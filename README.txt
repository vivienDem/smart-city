La classe CVM permet de lancer depuis Eclipse le projet en mono-JVM.

Pour un lancement multi-JVM, il faut ouvrir 6 terminaux dans le dossier
macrosoft. Puis executer les commandes suivantes en respectant l'ordre :

  terminal 1 : bash start-cyclicbarrier
  terminal 2 : bash start-gregistry
  terminal 3 : bash start-dcvm 1
  terminal 4 : bash start-dcvm 2
  terminal 5 : bash start-dcvm 3
  terminal 6 : bash start-dcvm 4
